# 介绍

目前来说，写了基础的一个框架，技术栈采用的是vite + vue3 + typescript。

## 项目

具体看package中的配置即可

* run

```
npm run dev
```

## 技术栈 

* 编程语言： typescript + JavaScript
* 构建工具： vite
* 前端框架： vue3
* 路由工具： vue-router
* 状态管理： pinia
* UI框架： element-plus
* CSS预编译： Sass
* 代码规范工具： editorConfig + prettier + eslint
